
#  Airbnb Data Analysis Dashboard

This project presents a data analysis of Airbnb listings using Google Looker Studio. The goal is to analyze occupancy rates, pricing trends, and revenue performance.

## Key Insights
- Occupancy rate decreases over time
- Shared rooms have higher occupancy
- Entire homes generate higher revenue
- Faster response time improves occupancy

## Tools
- Google Looker Studio
- Data visualization
- KPI analysis

This project demonstrates my ability to work with data, build dashboards, and generate business insights.
